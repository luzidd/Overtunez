# Overtunez

Additive synthesis engine for music production and sound design.
Written in Rust for performance and correctness.

## Features

- Additive synthesis with up to 256 partials
- Real-time modulation of partials
  - Amplitude
  - Phase
  - Frequency
- Built-in effects
- Multiple engines

## Formats

- CLAP
- VST3
- Standalone
